# Shipping Orders Performance Analysis (Excel Pivot Dashboard)
### by Karandeep

## Project Overview
This project focuses on analyzing a shipping orders dataset using **Microsoft Excel Pivot Tables and Pivot Charts**.  
The goal was to convert raw order data into a structured report and dashboard to identify trends in **orders, customer types, shipping modes, order priorities, and regional performance**.

By using Excel pivot-based reporting, the dataset was summarized into insights such as:
- Order distribution by **State and City**
- Customer segmentation by **Customer Type**
- Orders handled by different **Account Managers**
- Order patterns across **Order Priority**
- Performance comparison across **Shipping Modes**

## Key Pivot Table Analysis
The following Pivot Tables were created to analyze the dataset:

### 1) Orders by Region
- State-wise and City-wise order distribution
- Identifies top performing regions

### 2) Customer Type Analysis
- Orders distribution across customer categories
- Helps in customer segmentation

### 3) Account Manager Performance
- Number of orders handled by each account manager
- Useful for performance tracking

### 4) Order Priority Breakdown
- Categorization of orders by priority level (e.g., Critical, High, Medium, Low)
- Helps understand urgency trends

### 5) Shipping Mode Analysis
- Comparison of orders based on shipping modes
- Helps evaluate delivery preference trends
